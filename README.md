# AI Blockchain Voting System

A secure decentralized voting system combining AI-based facial verification with blockchain voting.

## Architecture

Face Recognition (Python)
        ↓
Backend Authorization
        ↓
Smart Contract (Solidity)
        ↓
Vote Recording on Blockchain

## Features

✔ Prevents double voting  
✔ Transparent vote counting  
✔ AI-based voter verification  
✔ Immutable blockchain records  

## Technologies

Solidity  
Ethereum / Polygon  
Python (Face Recognition)  
Web3.js# ai-blockchain-voting
